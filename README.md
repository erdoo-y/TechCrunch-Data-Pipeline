# TechCrunch-Data-Pipeline
A Python script that parses through the XML-based RSS-feed of a live Tech news website, extracts the latest news items in realtime and aggregates them into a CSV file.

#Features
* Data Cleaning: Removes rogue characters, parses XML namespaces and delivers tidy data.

#Technologies Used
* Python 3
* Urllib (HTTP client)
* XML Element Tree(XML parsing)
* CSV module(Data structuring)

#Sample Output:
The end result is an organized, structured CSV file that can be viewed/worked on as an Excel database with columns identified as 'Headline'| 'Link'| 'Date Published'| 'Category'| 'Author'

#What I learnt:
In the course of this project, I perfected the art of navigating nested XML trees, handling rogue characters and managing file system operations.
